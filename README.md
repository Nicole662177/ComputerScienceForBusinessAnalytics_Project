# ComputerScienceForBusinessAnalytics_Project


**How to use the code:**

<p>A version of Python3 is needed. In order to install all necessary modules, package management system PIP is used. </p>
<p>All of the commands below are for Windows.</p>
<p>To download <b>pip</b>:</p>
<p>  1. curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py</p>
<p>  2. python3 get-pip.py</p>
<p>Install <b>scipy</b>: <i>python3 -m pip install scipy </i></p>
<p>Install <b>scikit</b>: <i> pip install -U scikit-learn </i></p>
<p>Install <b>matplotlib</b>: <i>python3 -m pip install -U matplotlib</i></p>
  
<p>If everything went right, you can run the code from the command line by using the command: <i>python3 PythonApplication1.py</i></p>
